# ブログ記事管理システム

 Go v1.2.0 (Gin v1.9.0)
 React v18.2.0 (Next.js v13.1.6 MUI v5 TypeScript v5.1.3)

 Redis v8.11.5
 MySQL v1.5.0
 Docker v20.10.22
 
 【機能】
 ブログ記事閲覧画面（メイン）
 ブログ記事閲覧（編集確認モード）、新規作成、編集、消去
 会員情報新規登録、編集
 会員情報承認
